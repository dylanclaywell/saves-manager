## Renamed SavesManager → Pocket Quartermaster

App scope outgrew "saves manager" — RetroArch activity view shipped and SSD-to-handheld ROM curation is the next planned feature. New name carries a double pun (quartermaster = supply officer for an expedition; quarter = arcade coin). "Pocket" anchors it to handhelds.

### What changed (27 files + 2 new)
- **Code**: 3 `envPaths` callsites, marker filename (`.pqm-device-id.json`), `SAVESMANAGER_ALLOW_VIRTUAL_MOUNTS` → `PQM_ALLOW_VIRTUAL_MOUNTS`, HTTP User-Agent, UI nav + head title, devices page copy
- **Deploy**: install/update/run scripts (Linux + Windows), env-var prefix `SAVESMANAGER_*` → `PQM_*`, install dir `/opt/savesmanager` → `/opt/pqm`, env file `savesmanager.env` → `pqm.env`, systemd unit/service name `savesmanager` → `pqm`
- **CI**: build artifact `savesmanager-output.zip` → `pocket-quartermaster-output.zip`, release-please `package-name`
- **Docs**: README rewritten; release-notes-header rewritten
- **New files**: `deploy/pqm.service`, `deploy/pqm.env.example`, `server/plugins/migrate-legacy-data.ts`
- **Deleted**: `deploy/savesmanager.service`, `deploy/savesmanager.env.example`

### Migration shim
`server/plugins/migrate-legacy-data.ts` is a Nitro startup plugin. Renames the legacy parent dirs `%APPDATA%/savesmanager` and `%LOCALAPPDATA%/savesmanager` (and Linux equivalents) into their `pocket-quartermaster` counterparts on first boot. **Verified working on this machine** — both dirs successfully migrated during dev-server smoke test.

### Why rename the parent dir (not env-paths leaves)?
First version renamed `envPaths("savesmanager").config` (which is `<root>/savesmanager/Config`) into `<root>/pocket-quartermaster/Config` and ENOENT'd because the destination parent didn't exist. Switched to renaming `dirname(legacy.config)` → `dirname(current.config)` which moves the whole `<root>/savesmanager/` tree, no mkdir needed.

### Naming conventions
- Display name (UI, README h1, prose): **Pocket Quartermaster**
- Slug (repo, npm, env-paths): **pocket-quartermaster**
- Short alias (paths, env, service, marker): **pqm**
- Build artifact: **pocket-quartermaster-output.zip**
- HTTP UA: **PocketQuartermaster-Thumbnail/1.0**
